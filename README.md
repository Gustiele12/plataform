# plataform

Code to building a video conferencing platform similar to Zoom or Google Meet using WebRTC, but focused solely on video interactions (no microphone or screen sharing functionality).


project structure:
```
webrtc-video-platform/
├── public/
│   ├── index.html
│   ├── styles.css
│   ├── script.js
├── server.js
├── package.json
```
